Para Backend Django:

**python manage.py runserver**

el Backend por defecto estará en http://localhost:8000/

Para Frontend React:

**npm start**

el Frontend por defecto estará en http://localhost:3000/

Navega a http://localhost:8000/admin/ que es la pagina de superusuario admin para modificar el FrontEnd

usuario: mark

contraseña: 12345

Dentro de este url, se puede añadir películas, sesiones, etc..
Después de hacer cambios, navega a http://localhost:3000/ para ver los cambios.
